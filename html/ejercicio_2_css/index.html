¿Por qué aprender cerámica?

La cerámica no solo es una forma de expresión artística, sino también una práctica terapéutica que ayuda a reducir el estrés, mejorar la concentración y reconectar con el momento presente. Y además es un plan super cool para hacer con tu alguien que queres!
